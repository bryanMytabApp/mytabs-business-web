## React Template

# Tecnología
Front-end:
- React v18
- NodeJS v16.x
- AWS


# Configuración del ambiente de desarrollo
Requisitos:
- NodeJS v16.x
- NPM
- Serverless
- Python


# Instalación
- npm install
Instala las dependencias del cliente.

# Client
- npm run start
Corre el cliente en local. http://localhost:8080
